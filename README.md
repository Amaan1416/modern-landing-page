# Modern Landing Page

A modern responsive landing page built using **HTML, CSS and JavaScript**.  
This project demonstrates frontend development skills such as responsive layouts, UI components and interactive elements.

## 🚀 Features

- Responsive design (mobile friendly)
- Scroll reveal animations
- Sticky navbar with scroll effects
- Interactive pricing toggle (Monthly / Yearly)
- Modern UI cards and hover effects
- Clean and structured layout

## 🛠 Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM manipulation)
